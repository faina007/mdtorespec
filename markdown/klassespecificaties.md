# Klassespecificaties

## Object

|             |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Definitie   | Een fysiek, digitaal of conceptueel ding in de werkelijkheid dat van belang is voor een organisatie.                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Attributen  | [identificatie](#identificatie), [naam](#naam)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Toelichting | <ul><li>Het wordt aanbevolen om een naam te kiezen die betekenisvoller is voor anderen dan de direct betrokkenen. Bijvoorbeeld door een beknopte, specifieke omschrijving van de inhoud op te nemen. Denk hierbij aan een aanduiding van onderwerp, plaats en tijd. <li>MDTO schrijft geen maximale lengte voor, maar het wordt aanbevolen om een naam te kiezen van maximaal 80 karakters. Dit zorgt er bijvoorbeeld voor dat de naam in de presentatie van zoekresultaten volledig getoond kan worden. <li>Dit wordt, naast ‘naam’, ook wel eens ‘titel’ genoemd. </ul> |
| Voorbeelden | <ul><li>“Kapvergunning – Van de Spiegelstraat 12, Den Haag – 23 februari 2009” <li>Vergaderdossier van Wijkteam Zuid met huurderscommissies in Hoograven en Lunetten”<li>”Waalsdorpervlakte”<li>“Gemeente Zeewolde”                                                                                                                                                                                                                                                                                                                                                       |

## Actor

|              |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Definitie    | Persoon, groep, organisatie of functionaris.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Overerft van | [Object ](#object)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Attributen   | Geen                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Toelichting  | MDTO stelt geen nadere eisen aan de manier waarop een actor wordt vastgelegd. Idealiter wordt door middel van de identificatie van de actor verwezen naar een bestaande registratie, zoals het Actorenregister of de basisregistratie Handelsregister. In dat geval kan de naam van de actor ontleend worden aan die registratie. <br> <br> Merk op dat bij het vastleggen en verstrekken van persoonsgegevens over een actor beperkingen van toepassing zijn op grond van de Algemene verordening gegevensbescherming (AVG). MDTO doet geen uitspraken over welke persoonsgegevens mogen worden vastgelegd en beschikbaar gesteld op grond van de AVG. |
| Voorbeelden  | <ul><li>Een overheidsorganisatie<li>Een natuurlijk persoon<li>Een bedrijf</ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |

## Locatie
|              |                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Definitie    | Fysieke plaats in de ruimte.                                                                                                                                                                                                                                                                                                                                                                                                                          |
| Overerft van | [Object ](#object)                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Attributen   | Geen                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Toelichting  | MDTO stelt geen nadere eisen aan de manier waarop een locatie is vastgelegd. | Voorbeelden  | <ul><li>Een BAG-locatie<li>Een provincie<li>Een weg             </ul>                                                                                                                                                                                                                                                                                                                                                                                 |
